VFS Manager now comes in 2 flavors:

--------------------------------------------------------------------------------------------------------

VFSManager Stand-alone: Contains VFSManager v0.9.5.2 (BETA).  

	This version will work outside the ColdFusion administrator. 

--------------------------------------------------------------------------------------------------------

VFSManager CFADMIN: Contains VFSManager v0.9.5.2 (BETA).

	This version will not work outside the ColdFusion Administrator.  This version must be installed inside the ColdFusion Administrator to function.
